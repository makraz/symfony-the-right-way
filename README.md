# Symfony *The Right Way*

There's a lot of scattered information on the Web about Symfony â€” outdated tutorials, conflicting advice, and anti-patterns disguised as best practices that lead developers astray. **Symfony: The Right Way** is an easy-to-read, quick reference for Symfony developers, covering popular coding standards, recommended patterns, and links to authoritative resources around the Web.

*There is no single canonical way to build a Symfony application.* This guide aims to introduce new Symfony developers to topics they may not discover until it's too late, and to give seasoned developers fresh ideas on practices they've been following for years without ever reconsidering. It won't tell you which bundles or tools to use, but instead offers suggestions for multiple options, explaining the differences in approach and use-case when possible.

This is a living document and will continue to be updated with more helpful information and examples as they become available.
